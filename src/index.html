Other Projects

UofCoach

A web-app to connect tutors with tutees at the University of Calgary, developed in a group of 4 people for Calgary Hacks 2023 with the prompt of on-campus experience.

I focussed on developing the front-end using React with Typescript and Tailwind CSS. The backend is made using Flask (Python), MySQL and SQLAlchemy.

Skills used: React, TypeScript, Tailwind

Security with Stoffel!

I developed this web-app in a group of 5 people as the final project for my information security class. It presents illustrations of a red panda named Stoffel who asks quiz questions related to information security, and reacts differently depending of the user's input.

I was responsible for bringing this idea to fruition using HTML, CSS, and vanilla JavaScript. I also had a lot of fun collaborating with my group members!

Skills used: JavaScript, CSS, HTML
